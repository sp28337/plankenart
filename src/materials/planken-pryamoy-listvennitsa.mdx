Планкен прямой из лиственницы — один из наиболее востребованных материалов для вентилируемых фасадов в регионах с переменчивым климатом. Лиственница обладает редкой среди хвойных пород биостойкостью: при намокании древесина становится только плотнее, что исторически делало её материалом для строительства причалов и срубов в условиях постоянной влажности.

Прямой профиль без фасок даёт строгую, геометрически чистую поверхность. Это основа для монтажных раскладок любой ориентации — горизонтальной, вертикальной, диагональной и ёлочкой.
